⚠️Avalie cuidadosamente o risco antes de submeter ao CAB. Mudanças com risco alto ou crítico exigem aprovação da gestão sênior.
🚨O plano de rollback é obrigatório para mudanças de risco alto ou crítico. Documente exatamente como reverter a mudança se algo der errado.
📋O CAB (Change Advisory Board) analisa RFCs de mudanças normais. Mudanças padrão são pré-aprovadas; emergenciais passam pelo CAB pós-implementação.

⚠️A senha do SMTP deve ser inserida diretamente no Banco Console. Nunca armazene senhas no código.
Para Cloud Functions (firebase secrets):
firebase functions:secrets:set SMTP_HOST

SYSACK Agent Desktop

Instale nos computadores que deseja gerenciar

O SYSACK Agent Desktop transforma o SYSACK em um gerenciador completo de computadores:
✓ Inventario completo✓ Deploy de software ✓ Patch Management✓ Acesso remoto ✓ Gerenciar servicos✓ Terminal PS remoto ✓ Monitoramento CPU/RAM✓ Captura de tela
# Execute como Administrador:
Invoke-WebRequest -Uri "https://sysack.vercel.app/install-agent.ps1" -OutFile install.ps1
.\install.ps1 -Install
Requisitos: Windows 10/11 ou Windows Server 2016+ | Node.js 18+ | PowerShell 5+
O agente roda como Servico Windows (SYSTEM) e usa ~50MB de RAM.
